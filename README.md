# superstore-sales-analysis-powerbi
End-to-end data analysis project using SQL + Power BI with business insights and executive dashboards.
📊 Superstore Sales Analysis (SQL + Power BI)

🚀 Business Problem

Retail company wants to analyze sales, profit, and customer behavior to improve decision-making.

🛠️ Tech Stack

- SQL (Data Cleaning & Analysis)
- Power BI (Dashboard)
- Excel / CSV

📂 Project Structure

- data → cleaned dataset
- sql → analysis queries
- dashboard → Power BI file & visuals

📊 Key Insights

- Technology category generates highest profit
- Top customers contribute majority revenue
- High discounts reduce profitability
- Sales show steady YoY growth

📌 Recommendations

- Focus on high-margin products
- Reduce discounts on low-profit items
- Improve low-performing regions
- Retain high-value customers

📈 Business Impact

This dashboard helps stakeholders make data-driven decisions to improve profitability and growth.
